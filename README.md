# astrbot_plugin_bugua - 梅花易数时间起卦插件

AstrBot 卜卦插件，基于正宗梅花易数时间起卦法，以先天八卦序推演本卦、互卦、变卦，含五行体用生克分析与 AI 断解。

## 功能特性

- **梅花易数时间起卦** - 以农历年支、月、日、时辰为基础数，结合主题内容与用户身份，按先天八卦序起卦
- **三卦推演** - 本卦（现状）、互卦（过程）、变卦（结局）完整展示
- **五行体用生克** - 根据动爻判定体卦与用卦，分析五行生克关系，给出综合判断
- **AI 专业解读** - 调用 LLM 结合卦象与主题生成易学风格的专业断解
- **算式全程透明** - 起卦排盘公开展示农历时间、算式与六爻爻型
- **趣味随机建议** - "今天吃什么/穿什么/做什么"关键词触发，基于用户与日期的固定推荐

## 起卦原理

先天八卦序：1乾 2兑 3离 4震 5巽 6坎 7艮 8坤

变量说明：
- N = 农历年地支序数（子=1 ... 亥=12）
- M = 农历月
- D = 农历日
- T = 当前时辰序数（子=1 ... 亥=12）
- C = 主题内容哈希值（1-64）
- U = 用户ID哈希值（1-12）

公式：
```
上卦 = (N + M + D + C + U) % 8    余0取8
下卦 = (N + M + D + T + U) % 8    余0取8
动爻 = (N + M + D + T + C + U) % 6 余0取6
```

不同主题、不同用户、不同时辰均会产生不同卦象。

## 使用方法

### 安装

1. 将插件放入 AstrBot 的 `data/plugins/astrbot_plugin_bugua/` 目录
2. 安装依赖：`pip install borax>=4.0.0`
3. 重启 AstrBot

### 命令

```
卜卦 [主题]       主命令，起卦并给出解读
八卦 [主题]       同上（别名）
六爻 [主题]       同上（别名）
卜卦帮助          查看详细使用帮助
```

示例：
```
卜卦 期末考试
卜卦 近期财运
卜卦 感情
卜卦              不填主题则测综合运势
```

### 趣味功能

发送以下关键词会触发趣味推荐（同一用户同一天结果固定）：

```
今天吃什么
今天穿什么
今天做什么
```
