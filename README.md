# 🔮 中式卜卦 - AstrBot 卜卦插件

一个功能完整的中式传统卜卦插件，支持八卦、硬币、六爻等多种卜卦方式。

## ✨ 核心功能

### 卜卦方式

- **八卦卜卦** 📖 - 基于八卦的经典卜卦方式
  - 随机生成两个卦象
  - 提供详细的运势解读和建议
  
- **硬币卜卦** 🪙 - 模拟投掷硬币生成卦象  
  - 投掷三次硬币
  - 自动生成六爻卦象
  
- **六爻卜卦** ⚡ - 传统六爻卜卦
  - 生成六条爻
  - 支持变爻解读
  
- **塔罗式卜卦** 🎴 - 用八卦系统进行塔罗式解读
  - 支持单张卜卦
  - 支持过去-现在-未来三卡跨度
  
- **周运势预报** 📅 - 每日运势预测
  - 显示一周七天的运势
  - 每日运势等级和建议
  
- **生日运势预测** 🎂 - 根据生日预测运势
  - 按出生日期计算
  - 个性化的运势分析

## 📱 使用方法

### 安装

1. 将插件文件放入 `data/plugins/astrbot_plugin_pugua/` 目录
2. 重启 AstrBot 机器人
3. 插件自动加载

### 基本命令

#### 1. 八卦卜卦
```
/八卦           # 不指定主题的卜卦
/八卦 工作      # 针对工作的卜卦
/八卦 感情      # 针对感情的卜卦
/八卦 财运      # 针对财运的卜卦
```

#### 2. 硬币卜卦
```
/硬币           # 投掷硬币生成卦象
/硬币 爱情      # 针对爱情的硬币卜卦
```

#### 3. 六爻卜卦
```
/六爻           # 一般六爻卜卦
/六爻 升职      # 针对升职的六爻卜卦
```

#### 4. 周运势预报
```
/周运势         # 显示一周七天的运势
```

#### 5. 生日运势预测
```
/生日运势 2000-01-15    # 根据出生日期预测运势
```

#### 6. 塔罗式卜卦
```
/塔罗 single              # 单张卜卦
/塔罗 past_present_future # 过去-现在-未来三卡
```

#### 7. 帮助
```
/帮助           # 显示完整帮助信息
```

## 💡 使用建议

1. **参考而非绝对预言**：卜卦提供的是参考和指引，不是绝对的预言

2. **积极心态很重要**：运势会随着自己的努力而改变，相信积极心态的力量

3. **定期查看周运势**：建议定期查看周运势，把握每日机遇

4. **结合实际情况**：进行重大决策前，综合考虑卜卦结果和实际情况

5. **不必过度依赖**：卜卦是助力，不是拐杖，提高自身实力才是根本

## 🔍 工作原理

### 八卦卜卦流程

1. **随机选择两个卦象** - 代表主卦和变卦
2. **获取卦象信息** - 包括名称、符号、含义等
3. **运势评判** - 根据卦象综合判断运势等级
4. **生成建议** - 提供针对性的行动建议

### 硬币卜卦流程

1. **投掷三次硬币** - 每次返回正面(阳爻)或反面(阴爻)
2. **组合生成卦象** - 三条爻组成一个64卦中的一卦
3. **查询卦象含义** - 从卦象库中获取解读
4. **返回结果** - 显示卦象、运势和建议

## 📊 配置选项

插件支持以下配置（在 `config/settings.py` 中修改）：

```python
# 八卦卜卦配置
enable_bagua = True          # 启用八卦卜卦

# 六爻卜卦配置
enable_liuyao = True         # 启用六爻卜卦

# 塔罗卜卦配置
enable_tarot = True          # 启用塔罗卜卦

# 硬币卜卦配置
enable_coin = True           # 启用硬币卜卦

# 返回结果详细程度
detail_level = "详细"        # 可选: "简洁" 或 "详细"

# 是否包含建议
show_advice = True           # 显示卜卦建议

# 是否显示运势图
show_image = True            # 显示运势信息
```

## 📚 传统卜卦知识

### 易经八卦的含义

- **乾卦**：象征天，代表乾纲、刚健、权力、领导
- **坤卦**：象征地，代表柔顺、包容、厚德、承载
- **震卦**：象征雷，代表震动、兴奋、新生、发动
- **巽卦**：象征风，代表温和、渗透、进入、谦逊
- **坎卦**：象征水，代表危险、聪慧、流动、陷阱
- **离卦**：象征火，代表光明、美丽、聪惠、依附
- **艮卦**：象征山，代表停止、克制、沉着、凝聚
- **兑卦**：象征泽，代表喜悦、交流、柔和、破损
